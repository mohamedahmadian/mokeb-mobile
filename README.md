# موکب‌یار — اپ موبایل موکب‌دار

اپ آفلاین React Native (Expo) برای مدیریت موکب، زائرین، رزرو، حضور، وعده غذایی و امانات.

این پروژه **کاملاً جدا** از `admin-panel` و `backend` است و هیچ وابستگی به آن‌ها ندارد.

## اجرا

```bash
cd mobile-app
npm install
npm start
```

سپس با Expo Go یا شبیه‌ساز Android/iOS اجرا کنید.

## ساختار ناوبری

### نوار پایین (چپ به راست)
1. زائر
2. ورود و خروج
3. رزرو
4. وعده غذایی
5. پروفایل

### منوی بالا (سمت راست)
- داشبورد
- موکب‌ها
- امانات
- تغییر رمز عبور

## دیتابیس

SQLite محلی با ساختار نزدیک به `backend/prisma/schema.prisma`:

- `users`, `roles`, `user_roles`
- `mawkibs`, `mawkib_daily_inventory`
- `reservations`, `reservation_events`
- `meal_plans`
- `reservation_delivered_items`

## وضعیت فعلی

- ✅ ثبت‌نام / ورود / پروفایل / تغییر رمز (آفلاین)
- ✅ CRUD اولیه: موکب، زائر، رزرو، امانات
- ✅ ورود/خروج زائر + لیست حاضرین
- ✅ وعده غذایی امروز
- ⏳ فیلترهای پیشرفته، ویرایش جزئیات، تقویم ظرفیت (فاز بعد)

## نکته RTL

در اولین اجرا ممکن است برای اعمال کامل راست‌به‌چپ، یک‌بار اپ را ببندید و دوباره باز کنید.
